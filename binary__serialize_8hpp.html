Buf & chops::marshall ( Buf &  buf,
const T &  val 
)

Marshall a single arithmetic value or a std::byte into a buffer of bytes.

This is the "basic" marshall method, handling fundamental arithmetic values (char, short, int, double, float, etc) or a std::byte, as well as the customization point for user defined types. This function template calls a lower level function template that expands the buffer and appends the value to the buffer, performing byte swapping into big-endian format as needed. Single char and std::byte values will not be byte swapped.

Example usage - marshall an int as an unsigned 16 bit value:

std::vector<std::byte> buf;
// ...
marshall<std::uint16_t>(buf, my_int);

Example floating point usage - marshall a double:

std::vector<std::byte> buf;
// ...
marshall<double>(buf, my_double);
Template Parameters
CastValTypeThe fixed size type to be used for marshalling in the byte buffer, typically a type such as std::int32_t, std::uint32_t, std::uint16_t, float, etc; this type must always be supplied in the function call, since it is not deduced from the function arguments. Note that there are no standard typedefs for floating point types - use float or double as the casting type.
TType of value to be marshalled (typically deduced).
BufThe buffer type, which must contain an array of std::bytes, and must support size, resize, and data methods; chops::mutable_shared_buffer, std::vector<std::byte>, and chops::fixed_size_byte_array all satisfy this requirement.
Parameters
bufBuffer to store marshalled value, resize will be called to expand buffer.
valValue to be marshalled.
